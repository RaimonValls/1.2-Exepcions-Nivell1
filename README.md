-Description: This is a project with a "Producte" class containing name and price attributes, and a "Venda" class that manages a collection of products and calculates the total price. A custom exception, "VendaBuidaException," is implemented, which is triggered when the total for a sale with no products is calculated. An example of catching the "IndexOutOfBoundsException" exception is also included.
